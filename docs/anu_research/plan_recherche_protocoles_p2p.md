# Plan de Recherche : Protocoles P2P pour Messagerie Instantanée 2025

## Objectif
Analyser les protocoles peer-to-peer les plus adaptés pour la messagerie instantanée en 2025, avec focus sur WebRTC, libp2p, Matrix, et Signal Protocol.

## Critères d'Évaluation
- Performance (latence, throughput)
- Sécurité (chiffrement, authentification)
- NAT traversal (compatibilité avec différents types de NAT)
- Scalabilité (nombre d'utilisateurs, distribution)
